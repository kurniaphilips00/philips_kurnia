Instalasi Code igniter 4 :
1. Install laragon atau XAMPP
2. Download & install VSCode
3. Download & install CodeIgniter 4 : codeigniter.com
4. Extract & Copy file codeigniter ke folder xampp/htdocs atau laragon/www(cth ; cv)
5. Arahkan vscode ke folder cv, ubah env > .env; CI_ENVIRONMENT = production > CI_ENVIRONMENT = development
6. Konfigurasi database codeigniter :
	database.default.hostname = localhost
	database.default.database = quantum
	database.default.username = root
	database.default.password = 
	database.default.DBDriver = MySQLi
	database.default.DBPrefix =
	database.default.port = 3306
7. Ubah konfigurasi file php.ini > nyalakan extension.intl
8. Install composer untuk mempermudah pengembangan (instalasi phpspreadsheet, myth-auth, dompdf, ...dst.)
9. Di dalam terminal vs code ketik : "php spark serve", untuk memanggil server codeigniter
10.Ctrl+click pada "http://localhost:8080" atau ketik localhost:8080 pada Google atau Edge
